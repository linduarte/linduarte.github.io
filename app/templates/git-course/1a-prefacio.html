Bem-vindo ao Curso de Git

Este curso foi desenvolvido para te guiar do básico ao avançado no uso do Git, a ferramenta de controle de versão mais utilizada no mundo do desenvolvimento de software.

"O Git é como uma máquina do tempo para o seu código. Aprenda a usá-lo e nunca mais perca o histórico do seu trabalho!"

O que você vai aprender:

  • O que é Git e para que serve
  • Como instalar e configurar o Git
  • Comandos essenciais: init, add, commit, status, log, diff, branch, merge, clone, pull, push
  • Como trabalhar com repositórios locais e remotos
  • Fluxos de trabalho (workflows) e boas práticas
  • Como desfazer alterações e resolver conflitos
  • Integração com o GitHub

Como navegar pelo curso:

  1. Use o menu de navegação para acessar os tópicos.
  2. Marque seu progresso conforme avança.
  3. Leia, pratique e experimente os comandos no seu terminal.

Dica Rápida

Se você já tem experiência com Git, pode navegar livremente entre os tópicos. Se está começando, siga a ordem sugerida para melhor aproveitamento.
