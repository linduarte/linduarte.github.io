📚 Introdução ao Git

🚀 Capítulo 1: O que é Git?

Git é um sistema de controle de versão distribuído, criado por Linus Torvalds em 2005 para o desenvolvimento do kernel Linux.

🎯 Por que usar Git?
  • ✅ Rastreamento de mudanças: Acompanha todas as alterações no código
  • ✅ Trabalho em equipe: Múltiplas pessoas podem trabalhar no mesmo projeto
  • ✅ Backup automático: Histórico completo de todas as versões
  • ✅ Branches: Desenvolvimento paralelo de funcionalidades
  • ✅ Merge inteligente: Combina mudanças automaticamente
💡 Conceitos Fundamentais

📁 Repositório: Pasta que contém seu projeto + histórico Git

📸 Commit: "Foto" do estado do projeto em um momento

🌿 Branch: Linha de desenvolvimento independente

🔄 Merge: Combinar mudanças de diferentes branches
