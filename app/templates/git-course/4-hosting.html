Hospedagem - Projeto "Phonetic Website"

Seguiremos um roteiro baseado em um livro denominado "Practical Guide to Git and GitHub For Windows Users", escrito por Roberto Vormittag.

Etapas

  1. Etapa : Crie sua conta no GitHub
  2. Etapa : Faça 'login' em sua conta no GitHub.
  3. Etapa : Crie um repositório no GitHub - No menu do GitHub selecione o ícone 'Plus'(+) para criar um "New repository".
  4. Etapa : Dê um nome para este repositório - usaremos o nome 'simple-website'
  5. Etapa : Crie uma descrição para este projeto - "Phonetic alphabet website"
  6. Etapa : Selecione o repositório como 'Public'.
  7. Etapa : Marque a Caixa "Initialize this repository with a README file".
  8. Etapa : Deixe sem marcar o '.gitignore' e as 'license lists' como "None".
  9. Etapa : Pressione o botão "Create repository".
