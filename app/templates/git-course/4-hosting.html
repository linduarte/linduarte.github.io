Hospedagem - Projeto "Phonetic Website"

Seguiremos um roteiro baseado em um livro denominado "Practical Guide to Git and GitHub For Windows Users", escrito por Roberto Vormittag.

Etapas

Estas são as etapas essenciais para iniciar seu projeto no GitHub:

  1. Etapa: Crie sua conta no GitHub
  2. Etapa: Faça login em sua conta no GitHub.
  3. Etapa: Crie um repositório no GitHub usando o ícone "+" e selecione "New repository".
  4. Etapa: Nomeie o repositório como 'simple-website'
  5. Etapa: Descreva o projeto como "Phonetic alphabet website"
  6. Etapa: Selecione o repositório como 'Public'
  7. Etapa: Marque "Initialize this repository with a README file"
  8. Etapa: Deixe '.gitignore' e 'license' como "None"
  9. Etapa: Clique em "Create repository"
Voltar Próximo: connect
Para sair da sua conta, clique em Logout. Para registrar seu progresso, clique em Marcar como concluído.
