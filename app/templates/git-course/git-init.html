git init

O primeiro passo em um projeto com o Git é a inicialização do repositório onde se encontra o projeto.

A imagem acima mostra a criação de dois diretórios para o 'projeto-a'. Ao acionar o comando 'git init' o referido projeto passa a ser controlado pelo git. Observa-se ainda que aparece em destaque um símbolo de um 'ramo' e o nome dados a ele 'main'. O ramo é mais conhecido tecnicamente como 'branch'. Desta forma temos a certeza que foi inicializado o repositório do projeto-a.
